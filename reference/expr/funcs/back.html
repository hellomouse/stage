back

Syntax

(back count:number)

Return

true.

Description

back returns to a previous marker or non-blocking action, note that scene change also produces a point at the start of the scene. By default the count is set to 1 allowing you to simply do (back), the count is how many previous actions are skipped. In case you want to return to the latest action you may give a negative count, but that can be prone to creating infinite loops as back returns to the start of the action that called it.
