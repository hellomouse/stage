function type

The function type is a type containing a function name and its arguments. The function type is how you call functions in expressions, there are two forms for defining a function: (func arg1) and func(arg1). These differ in that the first form does not check if the function exists, while the second form does. The use for this is the form func-a(func-b) where func-b gets called only if func-a does not exist.
